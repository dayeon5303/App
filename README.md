Retro Weather & News Want_App

📌 프로젝트 소개
Want_App은 레트로 감성 UI를 기반으로, 핵심 기능에 집중하여 제작한 날씨·뉴스 조회 웹 애플리케이션입니다.
불필요한 요소를 배제하고 사용자가 필요한 정보를 직관적으로 확인할 수 있도록 화면 구성과 기능 설계에 중점을 두었습니다.
메인 화면에서는 명확한 버튼 구조를 통해 날씨 페이지와 뉴스 페이지로 이동할 수 있도록 설계했습니다.
마지막으로 각각 페이지를 확인하고 다시 메인으로 돌아 갈 수 있습니다. 

🌤 날씨 페이지

브라우저의 위치 정보를 활용하여 현재 위치 기반 날씨를 자동으로 표시합니다.
전 세계 도시 검색 기능을 구현하여 다양한 지역의 날씨를 확인할 수 있습니다.
날씨 상태에 따라 이모티콘이 동적으로 변경되도록 하여 시각적 직관성을 강화했습니다.
다른 도시 검색 후에도 현재 위치 날씨로 다시 돌아올 수 있도록 사용자 흐름을 고려해 기능을 구현했습니다.

📰 뉴스 페이지

API를 활용해 주요 기사를 불러오고, 한 화면에 한 개의 기사만 표시하도록 구성했습니다.
이전->다음 버튼을 통해 기사 탐색이 가능하도록 구현하여 가독성과 집중도를 높였습니다.
레트로 모니터 UI 콘셉트를 적용하여 디자인 일관성을 유지했습니다.

이 프로젝트는
사용자가 정말 원하는 기능만 담은 앱이라는 의미를 담아
Want_App(원앱-원해)이라는 이름으로 제작되었습니다.
