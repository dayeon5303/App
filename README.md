# 🌤 Retro Weather & News App

## 📌 프로젝트 소개
레트로 감성 디자인을 적용한 날씨 + 뉴스 조회 웹 애플리케이션입니다.  
현재 위치 기반 날씨 조회와 뉴스 기사 슬라이드 기능을 구현했습니다.

📱 모바일 / 태블릿 / PC 반응형 지원  
🎮 레트로 UI 디자인 적용  
🌍 OpenWeatherMap API 사용  
📰 News API 사용  

---

## 🛠 사용 기술

- React
- Redux Toolkit

- CSS (반응형 + 미디어쿼리)
- OpenWeatherMap API
- News API
- GitHub Pages 배포

---

## ✨ 주요 기능

### 🌤 날씨 페이지
- 현재 위치 기반 날씨 자동 조회 (다시 현재위치로 이동가능)
- 도시 검색 기능
- 온도 / 날씨 상태 / 아이콘 표시
- 반응형 모바일 가로모드 지원

### 📰 뉴스 페이지
- API 기반 뉴스 기사 불러오기
- 좌우 버튼으로 기사 이동
- 기사 이미지 + 제목 + 설명 표시
- 레트로 모니터 디자인 UI

### 🏠 홈 화면
- 애니메이션 효과 적용
- 버튼 클릭 시 페이지 이동
- 레트로 스타일 디자인
